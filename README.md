# 🚀 QR Code Generator

A modern and lightweight QR Code Generator built with **HTML, CSS, and JavaScript**. Generate QR codes instantly from text, URLs, phone numbers, email addresses, and more with a simple and user-friendly interface.

## ✨ Features

✅ Generate QR codes instantly

✅ Supports URLs, text, phone numbers, and emails

✅ Download QR codes as PNG images

✅ Custom QR code colors

✅ Clean and responsive design

✅ Fast and lightweight

✅ Beginner-friendly JavaScript project

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/your-username/QR-Code-Generator.git
```

### Navigate to the Project Folder

```bash
cd QR-Code-Generator
```

### Run the Project

Simply open `index.html` in your browser or use the VS Code Live Server extension.

---

## 📖 How to Use

1. Enter any text or URL.
2. Click **Generate QR Code**.
3. The QR code will be generated instantly.
4. Scan the QR code using any mobile device.
